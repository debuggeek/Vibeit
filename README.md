# Vibeit

A collection of vibe-related projects for testing and experimentation.

## Overview

Vibeit is a repository that houses various vibe-coated projects designed for testing different concepts, features, and vibes. Each project within this repository serves as an experimental playground to explore and refine ideas.

## Repository Structure

```
Vibeit/
├── projects/            # Collection of vibe-related test projects
│   ├── README.md       # Guide for the projects directory
│   └── ...             # Individual project directories
├── CLAUDE.md           # AI assistant guide for this repository
└── README.md           # This file
```

## Projects Directory

The `projects/` directory contains individual vibe-related projects. Each project is self-contained and focuses on specific testing goals or concepts.

For more information about the projects, see [projects/README.md](projects/README.md).

## Getting Started

1. Browse the `projects/` directory to find available projects
2. Each project has its own README with specific setup instructions
3. Follow the individual project documentation to get started

## Contributing

When adding new projects:
- Create a new directory under `projects/`
- Include comprehensive documentation
- Follow the guidelines in CLAUDE.md
- Keep projects self-contained and focused
